# odin-recipes

## Description
This is a basic HTML project from **The Odin Project** curriculum. The goal was to build a simple recipe website while learning the fundamentals of HTML, Git, and GitHub.

## Skills Demonstrated
* Structuring HTML documents with headings, paragraphs, and lists.
* Managing a local development environment using **WSL2**.
* Using Git for version control (add, commit, push) and troubleshooting GitHub privacy settings.
* Linking multiple pages and handling relative file paths for images.

## Reflections
Setting up the file structure between Linux (WSL) and Windows was a challenge, especially making sure the images were in the correct repository folder to be seen by the browser.
